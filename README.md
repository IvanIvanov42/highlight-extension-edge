# HighlightKit

Highlight specific text on any webpage. Useful for monitoring live feeds, tracking keywords, or making important info stand out.

## Features

- Highlights text in real-time as it appears on the page
- Three highlight colors: Yellow, Red, Black
- Case-insensitive search
- Works on any website
- Remembers your settings

## Installation

### Manual Installation

1. Download this repository
2. Open Edge and go to `edge://extensions/`
3. Turn on "Developer mode"
4. Click "Load unpacked" and select the folder

## How to Use

1. Click the extension icon
2. Type the text you want to highlight
3. Pick a color
4. Click "Add to Watch List"

That's it. The text will be highlighted on every page you visit.

## Privacy

All data stays in your browser. Nothing is sent anywhere.
